# Introduction
Everyone loves kittens! Have them on your terminal, artisan!

##Installation
###Step #1
```
composer require toniperic/kittens
```
###Step #2
Add `Toniperic\Kittens\KittensServiceProvider` to **providers** array in *config/app.php*

##Usage
- `php artisan meow`
- `php artisan meow:list`
